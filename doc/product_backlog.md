# Product Backlog

## Completed User Stories

| ID | Title | User Story | Priority | Status |
|---|---|---|---|---|
| US-01 | **Initial Prototype** | As a manager, I want basic inventory and menu management so that I can track items in my bakery. | High | **Done** |
| US-02 | **Modular Architecture** | As a developer, I want the codebase to be split into modules (Inventory, Menu, Cart) so that it is easier to maintain and extend. | High | **Done** |
| US-03 | **Itemized Cart** | As a customer, I want to see an itemized list of items I purchased in my cart. | Medium | **Done** |
| US-05 | **Basic Validation** | As a manager, I want the system to perform basic checks on menu selection to ensure correct items are added to the cart. | Medium | **Done** |
